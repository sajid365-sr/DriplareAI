⚙️ ১. প্রজেক্ট সেটআপ ও ক্লোন (Setup & Clone)

রিমোট রিপোজিটরি ক্লোন করা
GitHub বা রিমোট সার্ভার থেকে কোনো প্রজেক্ট প্রথমবার আপনার লোকাল কম্পিউটারে ডাউনলোড করতে এটি ব্যবহার করা হয়।
Git Command
git clone REMOTE_REPOSITORY_URL
নতুন লোকাল রিপোজিটরি চালু করা
একটি নতুন লোকাল ফোল্ডারে গিট ট্র্যাকিং বা ইনিশিয়ালাইজ করতে:
Git Command
git init

⚡ ২. বেসিক গিট ফ্লো (Basic Git Flow)

ফাইলের স্ট্যাটাস চেক করা
কোন কোন ফাইল পরিবর্তন করা হয়েছে, ডিলিট করা হয়েছে বা আন-ট্র্যাকড অবস্থায় আছে তা দেখতে:
Git Command
git status
ফাইল স্টেজিং এরিয়াতে যোগ করা
পরিবর্তনগুলো সেভ করার জন্য প্রস্তুত করা।
Git Command
# সব নতুন ও পরিবর্তিত ফাইল একসাথে যোগ করতে: git add . # নির্দিষ্ট কোনো ফাইল যোগ করতে: git add path/to/file.js
কোড সেভ বা কমিট করা
স্টেজিং এরিয়াতে থাকা ফাইলগুলোর একটি ভার্সন তৈরি করা।
Git Command
git commit -m "feat: add user authentication feature"
রিমোট গিটহাবে কোড পাঠানো (Push) এবং নিয়ে আসা (Pull)
আপনার লোকাল কোড রিমোটে পাঠাতে এবং রিমোটের কোড লোকালে নিয়ে আসতে:
Git Command
# গিটহাবে কোড পুশ করা git push # গিটহাব থেকে সর্বশেষ আপডেট টেনে নিয়ে আসা git pull

🌳 ৩. ব্রাঞ্চিং এবং সুইচিং (Branching & Switching)

ব্রাঞ্চ দেখা ও তৈরি করা
লোকাল ও রিমোট ব্রাঞ্চ দেখা এবং নতুন ব্রাঞ্চ তৈরি করা:
Git Command
# সব ব্রাঞ্চের লিস্ট দেখতে git branch -a # নতুন ব্রাঞ্চ তৈরি করে সরাসরি সুইচ করা git checkout -b feature/your-feature-name # তৈরি করা নতুন লোকাল ব্রাঞ্চ প্রথমবার রিমোটে সেটআপ করা git push -u origin feature/your-feature-name
ব্রাঞ্চ সুইচ ও ডিলিট করা
এক ব্রাঞ্চ থেকে অন্য ব্রাঞ্চে যাওয়া এবং ব্রাঞ্চ ডিলিট করা:
Git Command
# ব্রাঞ্চ সুইচ করা git checkout dev # লোকাল ব্রাঞ্চ ডিলিট করা (অন্য ব্রাঞ্চে থাকা অবস্থায় রান করতে হবে) git branch -d branch_name # রিমোট (GitHub) থেকে ব্রাঞ্চ ডিলিট করা git push origin --delete branch_name
লোকাল ব্রাঞ্চ মার্জ করা (Merge)
ফিচার ব্রাঞ্চের কাজগুলো dev বা main ব্রাঞ্চের সাথে মার্জ করা।
Git Command
# ১. dev ব্রাঞ্চে সুইচ করা git checkout dev # ২. ফিচার ব্রাঞ্চটি dev-এ মার্জ করা git merge feature/your-feature-name

📁 ৪. ডিরেক্টরি ট্রি এবং ফাইল ভিউ (Folder Structure)

Windows-এ ফোল্ডার ট্রি দেখা
ফোল্ডারের ভেতরের ফাইল এবং সাব-ফোল্ডার সুন্দর ট্রি আকারে দেখতে:
Windows CMD/PowerShell Command
# docs ফোল্ডারের ফাইল ও ফোল্ডার ভিউ tree docs /f # পুরো প্রজেক্টের ফাইল ও ফোল্ডার ভিউ tree /f
PowerShell ডিরেক্টরি ভিউ
PowerShell-এর মাধ্যমে প্রজেক্টের লিস্ট বা হিডেন ফাইল দেখা:
PowerShell Command
# হিডেন ফাইল ও ফোল্ডারসহ প্রজেক্টের রুট ফাইল লিস্ট দেখতে: Get-ChildItem -Force

📜 ৫. কমিট হিস্ট্রি ও লগ ট্র্যাকিং (Commit Logs & Diff)

কমিট হিস্ট্রি সংক্ষেপে বা গ্রাফ আকারে দেখা
লোকাল কমিট লগের সহজ ও গ্রাফিক্যাল ভিউ:
Git Command
# এক লাইনে কমিট হিস্ট্রি দেখা git log --oneline # ব্রাঞ্চিং স্ট্রাকচারসহ গ্রাফ আকারে দেখা git log --oneline --graph --all
লোকাল ও রিমোট ব্রাঞ্চের হিস্ট্রি তুলনা করা
লোকাল ও রিমোটের লেটেস্ট কোডের কমিটগুলোর অমিল চেক করা:
Git Command
# ১. রিমোটের ট্র্যাকিং ডাটা সিঙ্ক করা git fetch --all # ২. লোকাল dev ব্রাঞ্চে আছে কিন্তু রিমোট origin/dev ব্রাঞ্চে (GitHub) এখনো পুশ করা হয়নি এমন কমিট দেখতে: git log origin/dev..dev --oneline # ৩. রিমোটে আছে কিন্তু আপনার লোকালে এখনো পুল করা হয়নি এমন কমিট দেখতে: git log dev..origin/dev --oneline

🔑 ৬. কমিট হ্যাশ ও ফাইল ডিটেইলস (Commit Hashes)

নির্দিষ্ট ফাইলের কমিট হিস্ট্রি ট্র্যাক করা
একটি নির্দিষ্ট ফাইল অতীতে কখন কোন কমিটে মডিফাই বা ডিলিট করা হয়েছে তা খুঁজে বের করতে:
Git Command
git log --all --full-history -- "path/to/file.js"
কমিটের বিস্তারিত কোড পরিবর্তন দেখা
সুনির্দিষ্ট কমিট হ্যাশ দিয়ে ওই কমিটের সব কোড পরিবর্তন দেখা:
Git Command
# লেটেস্ট কমিট হ্যাশ দেখতে: git rev-parse HEAD # কমিটের ভেতরের কোড পরিবর্তন দেখতে: git show COMMIT_HASH

🛠️ ৭. স্ট্যাশ ও কোড রিকভারি/আনডু (Stash & Recovery)

গিট স্ট্যাশ (Stashing) — কাজ লুকিয়ে রাখা
আন-কমিট করা ফাইলগুলো সেভ না করেই সাময়িকভাবে লুকিয়ে রাখতে এবং পরবর্তীতে ফেরত আনতে:
Git Command
# চলমান পরিবর্তনগুলো লুকিয়ে রাখা git stash # লুকিয়ে রাখা কাজ ফিরিয়ে আনা git stash pop
ভুল মার্জ বা রিসেট আনডু করা (Git Reflog)
ভুলবশত কোড ডিলিট বা রিসেট করে ফেললে গিট রিফ্লগ ব্যবহার করে আগের অবস্থায় ফিরে যাওয়া যায়:
Git Command
# লোকাল অ্যাকশনের রেফারেন্স লগ দেখা git reflog # পাওয়া আইডি (যেমন HEAD@{2}) দিয়ে সেই অবস্থায় ফিরে যাওয়া git reset --hard HEAD@{2}
মার্জ করা বাতিল করা
মার্জ কনফ্লিক্ট দেখা দিলে সম্পূর্ণ মার্জিং প্রসেস বাতিল করে পূর্বের স্বাভাবিক অবস্থায় ফিরতে:
Git Command
git merge --abort

🌐 ৮. রিমোট কনফিগারেশন ম্যানেজমেন্ট (Remote Config)

রিমোট URL দেখা ও পরিবর্তন করা
আপনার লোকাল প্রজেক্টটি কোন GitHub লিংকের সাথে সংযুক্ত আছে তা দেখতে বা পরিবর্তন করতে:
Git Command
# সংযুক্ত রিমোট URL দেখতে: git remote -v # নতুন রিমোট URL সেটআপ করতে (যেমন রিমোট সরানোর পর): git remote add origin REMOTE_REPOSITORY_URL # বিদ্যমান রিমোট URL পরিবর্তন করতে: git remote set-url origin REMOTE_REPOSITORY_URL
